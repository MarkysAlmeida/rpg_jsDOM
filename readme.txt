Mini RPG em JavaScript
  Este projeto é um pequeno sistema de batalha RPG feito em JavaScript para treinar conceitos básicos de programação e manipulação do DOM.

📚 O que eu aprendi

- Com esse projeto eu pratiquei vários conceitos importantes de JavaScript:
- Variáveis (let e const) para armazenar dados do jogador e do inimigo.
- Objetos para representar personagens, com propriedades como nome, vida e ataque.
- Arrays para guardar vários tipos de inimigos e escolher um aleatoriamente.
- Funções para organizar a lógica do jogo, como atacar, criar inimigos e atualizar a tela.
- Manipulação do DOM usando document.getElementById() para mostrar informações do jogo no HTML.
- Uso de Math.random() para criar ataques e inimigos aleatórios.
- Estruturas condicionais (if) para controlar regras do jogo, como vitória ou derrota.

🎮 Como o jogo funciona
- O jogador escolhe uma classe (Guerreiro ou Mago).
- Um inimigo aleatório aparece.
- O jogador ataca o inimigo.
- Se o inimigo sobreviver, ele contra-ataca.
- A batalha continua até que um dos dois fique sem vida.

🎯 Objetivo do projeto
- O objetivo foi praticar lógica de programação e JavaScript básico, simulando um sistema simples de combate de RPG.
